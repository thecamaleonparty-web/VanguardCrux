# Fintech Case Study Images

This directory should contain the following images for the Business Case Analyzer Pro case study:

## Required Images (WebP format):

1. **fintech-hero-ui.webp** - Main hero background showing the application UI
2. **fintech-code-snippet.webp** - Code snippet showing technical implementation
3. **fintech-mobile-grid.webp** - Mobile responsive view
4. **fintech-pdf-export.webp** - PDF export functionality

## Image Guidelines:
- Format: WebP
- Dimensions: At least 1920x1080 for hero images
- Optimize for web performance
- Dark theme compatible (the site uses dark backgrounds)
